# Página Breaking Dance
Site desenvolvido sobre o Breaking Dance nas Olimpíadas de 2024, com foco em prática de HTML e CSS.

✨ Sobre o Projeto

Este projeto apresenta informações sobre a inclusão do breaking como modalidade olímpica, destacando sua história, relevância cultural e presença nos jogos.

🛠️ Tecnologias Utilizadas

🔹 HTML5

🔹 CSS3

🎯 Objetivo

Praticar conceitos fundamentais de desenvolvimento web, como:

🔹 Estruturação de páginas com HTML

🔹 Estilização com CSS
